## 🚀 核心项目演练 | Projects

### 👁️ Project InsightEye `2025.10 - Present`
> **企业级技术情报聚合与自动化报告生成平台**
* **项目定义**: 针对企业级痛点开发的“重型”智能化信息处理平台。
* **核心架构**: 基于高级 RAG 检索增强技术与自定义多 Agent 协同流，打通了从海量网络技术源（如 `gpt-researcher` 深度检索演进）到结构化研报输出的全链路。
* **技术亮点**: 纯手工优化了 Agent 之间的异步通信与状态管理，极大地降低了长文本上下文的幻觉率，实现了真正具备商业可用性的自动化研报生成。

### 🤖 机器人集群视觉退化分析 `Research Project`
> **基于 YOLOv8 的硬核学术实验研究**
* **项目定义**: 在研究生期间主导的自动化视觉感知实验，用以检测硬件层面的传感器失效。
* **硬核攻关**: 拒绝合成模拟数据！完全在物理层面手动对机器人（R5, R6, R7 集群）实施精密相机镜头模糊处理，以还原最真实的物理降级场景。
* **数据闭环**: 通过 YOLOv8 精密捕捉目标检测指标的动态衰减，清洗并重建机器人运动轨迹数据集，实现对视觉传感器退化程度的量化评估。
